Sprite system v3
Sprite ROM holds 256 sprite definitions. VRAM holds 256 sprite attributes, 4 colour palettes, 2 tile tables and 2 sprite tables.

Sprite definition: 32B, 8x8 pixels
	Pixel: 8b
		* Index into palette: 4b
		* Unused: 4b
Colour palette: 16B, 16 colours
	Colour 1B:
		* Red: 3b
		* Green: 3b
		* Blue: 2b
Sprite table: 320B, 64 sprite entries
	Sprite entry: 5B
		* X coord: 1B
		* Y coord: 1B
		* Sprite no.: 1B
		* Current sprite no. increment: 2b
		* Current x increment: 2b
		* Current y increment: 2b
		* Colour palette: 2b
		* Attribute no.: 1B
		Sprite attribute: 1B
			* Max no. increment: 2b
			* Max x increment: 2b
			* Max y increment: 2b
			* Every fourth update too: 1b
			* Enabled: 1b

Tile table: 1800B, 30x30 tile entries
	Tile entry 2B:
		* Colour palette: 2b
		* Enabled: 1b
		* Unused: 5b
		* Sprite no.: 1B

Tools:
    * PNG files to sprite ROM
